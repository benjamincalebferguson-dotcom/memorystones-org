# Memory Stones

Static landing page for `memorystones.com.au`.

## Project Structure

- `index.html` - main site page with inline styles and scripts
- `impact_image.jpg`
- `kit_image.png`
- `lady_holding_stone_.jpeg`
- `stone_on_bench_.jpeg`
- `stones_on_shelf.jpeg`
- `netlify.toml` - Netlify deploy configuration

## Local Preview

Because this is a plain static site, you can preview it by opening `index.html` in a browser.

## Netlify Deployment

This repo is configured for a buildless Netlify deploy.

- Build command: none
- Publish directory: `.`

If connecting the GitHub repo in Netlify, make sure the site publishes from the repository root.
